# LearnC

Learn C by building real programs.

## What's Here

Right now: a software renderer that draws to a window. More projects coming later.

## Examples

### Renderer
Software rendering from scratch - everything runs on CPU, no GPU.

**Examples:**
- 01_basic_shapes - 2D primitives and animation
- 02_3d_cube - wireframe cube with depth
- 03_3d_pyramid - solid shapes with lighting
- 04_particles - physics simulation
- 05_textured_cube - texture mapping
- 06_simple_game - simple game demo
